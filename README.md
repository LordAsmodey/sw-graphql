# React + TypeScript + Vite + GraphQL + Apollo Client + GraphQL Code Generator + Next UI + Automatic deployment to GH Pages

### Local installation:

1. Clone repository.

2. Install deps
```shell
yarn
```
3. Run project.
```shell
yarn run dev
```

### Other comand:
To run the GraphQL Code Generator watcher
```yarn run gql-w```

Manual Deploy to GH Pages
```yarn run deploy```
